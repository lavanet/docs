## Parameters

| Parameter          | Required/Optional | Description                                                                                                                                                  |
|--------------------|-------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `privateKey`       | **Required**       | The private key of the staked Lava client for the specified chainID.                                                                                         |
| `chainId`          | **Required**       | The ID of the chain to query. Refer to `lavad q spec show-all-chains` for chain IDs.                                                                               |
| `pairingListConfig`| Optional          | Specifies the Lava pairing list config for communication with the Lava network. If not provided, the default list is used.                                   |
| `networkId`        | Optional          | Chain ID of the network to query. Defaults to the matching chain ID.                                                                                        |
| `geolocation`      | Optional          | Geolocation to be used. Default is 1 for North America providers. Use 2 for EU providers.                                                                   |
