# All Blocks (Vercel App)

**All Blocks**是一个使用 "create-next-app "启动的[Next.js](https://nextjs.org/)项目，可从 Lava 上多个受支持的链中抓取最新区块，实时测量并报告延迟和正常运行时间信息。
