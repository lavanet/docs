# 运行 Lava RPC 节点

## 配置要求 📄 

当前推荐的运行Lava RPC节点的[硬件配置条件](/reqs)如下:

```
16 GB RAM
512 GB SSD
x64 2.0 GHz 4v CPU 
```

## 安装 Lava 🌋

使用[安装指南](/install-lava)安装所有二进制文件，特别是`lavad`:

```bash
git clone https://github.com/lavanet/lava.git
cd lava
make install-all
```

## 启动节点 🚀

根据可用的[guides](/testnet)设置和配置您的节点。
您可以使用`Cosmovisor`(推荐)或手动设置节点。
